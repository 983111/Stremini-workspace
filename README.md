# Stremini Workspace

## Introduction

Stremini Workspace is a unified agent console that brings together a suite of specialized AI agents for research, personal organization, finance, startup/legal compliance, code generation, growth strategy, AI system architecture, and data analytics.

## Agents

- **Research & Math Agent** – Generate structured research papers, solve advanced math problems, and organize technical analysis.
- **ARIA Personal OS** – Your strategic second brain for planning, habits, reflection, and long‑term personal operating system decisions.
- **Finance Agent** – Build financial models, evaluate business metrics, and receive concise analysis for business and investment decisions.
- **Startup & Legal Compliance Agent** – Navigate startup setup, policy readiness, compliance checkpoints, and legal process guidance.
- **Code Agent** – Design architecture, generate production‑grade code, and iterate quickly on implementation.
- **Growth & Marketing Intelligence Agent** – Build GTM strategy with competitor research, ICP modeling, SEO planning, funnel diagnostics, and structured outputs like SWOT, TAM/SAM/SOM, and KPI summaries.
- **AI Systems Architect Agent** – Design production AI systems with RAG architecture, vector database selection, model trade‑off analysis, orchestration design, and scaling‑aware cloud planning.
- **Data & Decision Intelligence Agent** – Interpret CSV and JSON business data with retention, conversion, and anomaly analysis, then produce clear insights, risk flags, hypotheses, and experiment priorities.

## How to Use

1. **Clone the repository**  
      git clone https://github.com/983111/Stremini-workspace.git
   cd Stremini-workspace
   
2. **Open the workspace**  
   Double‑click `index.html` or open it in a browser:  
   `file:///path/to/Stremini-workspace/index.html`

3. **Select an agent**  
   Click the card for the desired agent. The agent page will load in the same window.

## Development

The project consists of static HTML, CSS, and JavaScript files. To add a new agent:

1. Create a new HTML file (e.g., `NewAgent.html`).
2. Add a card entry in `index.html` linking to the new file.
3. Follow the existing styling conventions in `index.html` for consistency.
